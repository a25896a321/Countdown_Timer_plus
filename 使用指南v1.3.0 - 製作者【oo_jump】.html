1. 程式簡介

倒數計時器是一款專為需要精準時間掌控的使用者設計的工具， 特別適合遊戲攻略、直播計時、技能冷卻管理等使用情境。 支援同時管理多組獨立計時器、全螢幕懸浮顯示、全域熱鍵觸發，讓你在不離開遊戲的情況下掌控每一個倒數節點。

主要特色

特色說明
多組計時器最多可同時設定數十組獨立倒數計時器，同時在畫面上顯示
全域熱鍵在任何視窗前景下（包含遊戲）均可觸發，無需切換視窗
懸浮窗模式全螢幕遊戲不受干擾，計時圖示始終顯示於最上層
圖示顯示每組計時器搭配自訂圖片，視覺上一目了然
音效提示可設定倒數結束前幾秒提前播放提示音效
設定檔管理最多 10 組設定可隨時切換，支援匯入匯出分享
多語言支援繁體中文 / English 即時切換

2. 系統需求與安裝

系統需求

項目需求
作業系統Windows 10 / 11（64位元）
建議解析度1920×1080 以上
其他建議以系統管理員身分執行，確保全域熱鍵正常運作

安裝方式

  1. 將整個 Countdown_Timer 資料夾放置於任意位置。
  2. 直接雙擊 Countdown_Timer_plus.exe 執行。
  3. 無需安裝 Python 或任何額外套件。

目錄結構

Countdown_Timer_plus/
├── Countdown_Timer_plus.exe  ← 主程式（雙擊執行）
├── sound_type/               ← 音效檔案資料夾（可自行新增 mp3/wav）
└── png_type/                 ← 圖片檔案資料夾（可自行新增圖片）
請勿移動或刪除 sound_type 和 png_type 資料夾，程式依賴此相對路徑讀取資源。

【已停用】程式整體已停用，所有熱鍵與計時暫停
尚未載入設定檔目前無作用中的設定檔，請先套用一個

⚙️ 計時管理開啟多組計時器設定視窗

主視窗中央顯示目前設定檔內所有計時組，每一列顯示：

  • 快捷鍵 — 觸發此計時的按鍵，例如 [Numpad/]
  • 倒數數字 — 即時顯示剩餘秒數（大字體）
  • 模式 — 自動循環 ／ 倒數後停止 ／ 雙回合切換
  • 回合標示 — 雙回合模式下顯示「回合一」或「回合二」

3.4 透明度拉條

位於視窗底部，可即時調整主視窗的透明度（0.1 幾乎全透明 ～ 1.0 完全不透明）。

4. 設定檔管理

點擊 💾 設定檔 按鈕開啟管理視窗。視窗左側為插槽列表，右側顯示詳情與操作按鈕。

4.1 設定檔插槽（10 組）

程式提供 10 個設定檔插槽（No.1 ～ No.10），每個插槽可獨立儲存一套完整的計時組設定。
點選任意插槽，右側詳情欄會顯示該設定檔的所有計時組資訊（名稱、快捷鍵、模式等）。

  • 有 【預設】 標示的插槽，程式啟動時會自動載入
  • 空插槽顯示為「（空）」

4.2 套用、匯入、匯出、清空

按鈕說明
✅ 套用將選取的設定檔套用為目前使用的設定，立即生效
📥 匯入從 .json 檔案匯入設定（會覆蓋選取的插槽）
📤 匯出將選取插槽的設定儲存為 .json 檔案（備份或分享用）
🗑 清空清除該插槽所有計時組設定（操作不可復原）
✏️ 編輯名稱修改設定檔的顯示名稱

4.3 預設設定檔

插槽 1（No.1）固定為預設設定檔，程式啟動時自動載入插槽 1 的設定。
若插槽 1 為空，啟動時將顯示「尚未載入設定檔」，請手動套用所需插槽。

建議將最常用的設定存放於插槽 1，即可省去每次手動套用的步驟。

4.4 程序快捷鍵設定

設定檔管理視窗的下半部提供三個程序快捷鍵（非計時器啟動）的設定：

功能預設按鍵說明
全部暫停F6所有計時器回到初始時間並停止倒數
啟用／停用F7切換程式整體啟用狀態
懸浮框開關F8切換懸浮窗模式（開啟或關閉懸浮窗）

設定方式： 點擊對應輸入框（輸入框變黃色）→ 按下想要的按鍵 → 自動完成設定。

支援一般鍵盤按鍵、Numpad 數字鍵、功能鍵（F1–F12）、特殊鍵（CapsLock、`、Numpad* 等）。

5. 多組計時管理

點擊 ⚙️ 計時管理 開啟設定視窗。視窗左側為計時組列表，右側為詳細設定欄。

5.1 新增 / 刪除 / 排序計時組

按鈕說明
➕ 新增在列表末端新增一個預設計時組
⬆ 上移將選取的計時組向上移動一格
⬇ 下移將選取的計時組向下移動一格
🗑 刪除刪除選取的計時組（不可復原）

點擊左側列表中的計時組名稱，右側會顯示該組的完整設定供編輯。

5.2 計時名稱與快捷鍵

  • 計時名稱 — 自由填寫，顯示於主視窗計時列與懸浮窗圖示下方
  • 快捷鍵 — 點擊輸入框後按下任意按鍵即完成設定（支援 Numpad 鍵）
快捷鍵採用全域監聽，即使遊戲視窗在前景仍可觸發，無需切換視窗。

5.3 倒數模式

每個計時組可選擇三種倒數行為：

🔄

自動循環倒數

倒數歸零後自動重新開始，無限循環，適合週期性機制

⏹

倒數後停止

倒數歸零後停止並顯示結束，等待下次手動觸發，適合技能冷卻

⟳

雙回合切換

設定兩個時間，自動交替倒數，適合兩種狀態輪替的機制

雙回合範例：機制A持續 40 秒 → 機制B持續 30 秒 → 回到機制A…如此循環。

5.4 快捷鍵功能

按下快捷鍵時的觸發行為（可從兩種選項中選擇）：

功能說明
重置並開始無論目前狀態，立即重置計時並重新開始倒數（雙回合模式會回到回合一）
開關（關閉會重置）未倒數中 → 開始；倒數中 → 重置並停止（適合需要臨時取消的情境）

5.5 圖示效果模式

控制計時圖示的顯示效果，三種模式說明如下：

模式倒數中最後 3 秒倒數結束適用情境
預設彩色原圖彩色↔灰階閃爍灰階一般計時（最後3秒警示）
冷卻模式灰階（冷卻中）灰階↔彩色閃爍彩色原圖（就緒）技能冷卻計時
僅用原圖彩色原圖彩色原圖彩色原圖純視覺標記，不需灰階效果
冷卻模式適合技能冷卻計時：倒數中顯示灰階表示「技能冷卻中」，倒數結束後顯示彩色表示「技能就緒」，直覺易懂。

5.6 圖片設定

  • 從下拉清單選擇 png_type 資料夾中的圖片
  • 🔎 瀏覽 — 點擊後在按鈕旁浮現圖片預覽視窗
  • 📂 開啟資料夾 — 直接以檔案總管開啟 png_type 資料夾

5.7 音效設定

設定項目說明
音效選擇從 sound_type 資料夾中選擇 .mp3 / .wav 音效
播放方式：單次播放在「倒數結束前 N 秒」時播放一次
播放方式：分段播放按設定次數，在倒數過程中平均分散播放
提前幾秒在倒數結束前幾秒觸發音效（例如設定 2 = 剩 2 秒時播放）
分段次數分段播放模式下，音效共播放幾次
▶ 測試立即播放所選音效，確認效果
📂 開啟資料夾以檔案總管開啟 sound_type 資料夾

6. 懸浮窗設定

點擊 🖥 懸浮窗設定 開啟設定視窗，調整完成後點擊 ✅ 套用（即時生效）或 💾 儲存（持久保存）。

6.1 位置與大小

設定說明
X / Y懸浮窗初始出現的螢幕座標（左上角為 0,0）
圖示大小（像素）每個計時圖示的顯示尺寸（建議 48 ～ 96 px）
圖示間距（像素）相鄰計時組之間的水平間距
懸浮框透明度整體透明度（0.1 幾乎全透明 ～ 1.0 完全不透明）

6.2 顯示控制（勾選框）

選項預設說明
顯示拖曳手把✓ 開啟懸浮窗左上角的移動圖示，長按可拖曳移動懸浮窗位置
顯示設定齒輪✓ 開啟懸浮窗左上角的 ⚙ 按鈕，點擊可返回主視窗
顯示背景底色✗ 關閉是否顯示懸浮窗的整體背景色塊
顯示數字背景✗ 關閉倒數數字是否加背景色塊
顯示名稱背景✓ 開啟計時名稱文字是否加背景色塊
隱藏未使用倒數✗ 關閉勾選後，未啟動的計時組自動隱藏，倒數開始後才出現
顯示圖片✓ 開啟是否顯示計時圖示；關閉後僅顯示倒數數字與計時名稱，適合純文字模式

6.3 顏色自訂

點擊各顏色方塊可開啟系統色盤選色：

顏色設定說明
背景顏色懸浮窗背景底色（需勾選「顯示背景底色」才可見）
數字背景顏色倒數數字的背景色（需勾選「顯示數字背景」才可見）
名稱背景顏色計時名稱的背景色（需勾選「顯示名稱背景」才可見）
數字顏色倒數秒數的字體顏色
名稱顏色計時名稱的字體顏色

6.4 名稱顯示設定

名稱顯示位置（5 種選項）：

選項效果
圖片上方（外）名稱顯示於圖示正上方，圖示下方有倒數數字
圖片下方（外）名稱顯示於圖示正下方（預設）
圖片內中間名稱文字疊加於圖示的中央位置
圖片內上方名稱文字疊加於圖示的上緣
圖片內下方名稱文字疊加於圖示的下緣

名稱對齊方式（3 種選項）：靠左 / 置中（預設） / 靠右

7. 懸浮窗操作

點擊主視窗的 ⇄ 切換懸浮模式 按鈕，或按 F8（預設），進入懸浮窗模式。

操作功能
長按拖曳手把圖示移動懸浮窗至螢幕任意位置（新位置自動儲存）
點擊 ⚙ 齒輪圖示返回主視窗（懸浮窗同時關閉）
按 F8（預設）切換回主視窗 / 重新顯示懸浮窗

懸浮窗顯示說明

  • 永遠顯示於所有視窗最前方（全域置頂）
  • 計時圖示由左至右橫向排列，每組間距可在設定中調整
  • 每個圖示上方顯示即時倒數秒數（大數字）
  • 圖示下方（或其他設定位置）顯示計時名稱
  • 倒數最後 3 秒時圖示會開始閃爍警示（依圖示效果模式）

8. 程序快捷鍵說明

程序快捷鍵（非計時器啟動）在程式停用狀態下仍然有效，確保你隨時都能重置或開關懸浮窗。
按鍵（預設）功能備註
F6全部暫停所有計時器回到初始時間並停止倒數
F7啟用／停用停用時計時停止，啟用後可重新開始使用
F8懸浮窗開關切換懸浮窗顯示模式
（各計時組自訂鍵）觸發對應計時器依設定的「快捷鍵功能」執行重置或開關

以上預設按鍵可在 💾 設定檔 視窗的「程序快捷鍵設定」區塊中自訂更改。

9. 自訂圖片與音效

9.1 新增圖片

  1. 將圖片（.png / .jpg / .jpeg / .gif）放入 png_type 資料夾
  2. 可在 png_type 內建立子資料夾分類，程式會自動遞迴掃描
  3. 重新開啟計時管理設定視窗，新圖片即出現在下拉清單中

9.2 新增音效

  1. 將音效（.mp3 / .wav）放入 sound_type 資料夾
  2. 可在 sound_type 內建立子資料夾分類
  3. 重新開啟計時管理設定視窗，新音效即出現在下拉清單中
建議使用清晰、簡短的語音提示音效，搭配「提前幾秒」設定，達到最佳提示效果。

10. 常見問題

Q：全域熱鍵沒有反應？

A：請以「系統管理員身分」重新執行程式。右鍵點擊 Countdown_Timer_plus.exe → 以系統管理員身分執行。

Q：音效無法播放？

A：確認音效檔案格式為 .mp3 或 .wav，且已放置於 sound_type 資料夾內。可使用「▶ 測試」按鈕確認。

Q：懸浮窗跑到螢幕外找不到？

A：在 🖥 懸浮窗設定 中，將 X/Y 座標改回螢幕內的數值（例如 X=500, Y=700），再按「✅ 套用」即可。

Q：設定檔不見了？

A：設定儲存於程式同目錄下的 countdown_config.json。若不小心刪除，程式會自動生成內建預設設定。

Q：如何備份我的設定？

A：使用 💾 設定檔 → 📤 匯出，將設定儲存為 .json 檔案，可備份也可分享給其他人使用。

Q：程式更新後設定會消失嗎？

A：不會。只要 countdown_config.json 與新版 Countdown_Timer_plus.exe 放在同一資料夾，設定即自動繼承。

Q：計時器精度如何？

A：程式每 0.3 秒更新一次，顯示秒數採四捨五入，時間計算使用系統時間戳差值，避免誤差累積，精度可達 ±0.3 秒。
